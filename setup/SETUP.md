# Тапсырыс жүйесін қосу — нұсқаулық

Жүйе қалай жұмыс істейді:

```
Клиент форманы толтырады
   ├─→ Google Sheets кестесіне жазылады  (Excel-кесте, барлық тапсырыс)
   ├─→ Сізге Telegram-ға хабар келеді    (автоматты)
   ├─→ Клиенттің WhatsApp-ы ашылып, тапсырыс мәтіні сізге жіберуге дайын тұрады
   └─→ admin.html — логин/парольмен барлық тапсырысты көресіз
```

Орнату ~15 минут алады. Бәрі тегін.

---

## 1-қадам. Google Sheets кестесін жасау

1. [sheets.new](https://sheets.new) ашыңыз (Google аккаунтпен).
2. Кестені «Құлыншақ тапсырыстар» деп атаңыз.

## 2-қадам. Apps Script қосу

1. Кестеде: **Extensions → Apps Script**.
2. Ашылған редактордағы бар кодты өшіріп, осы репозиторийдегі
   **`setup/apps-script.gs`** файлының кодын толық көшіріп қойыңыз.
3. Кодтың басындағы `ADMIN_KEY` мәнін өзіңіз ойлап тапқан парольге өзгертіңіз —
   бұл админ-панельдің паролі болады.
4. **Сақтаңыз** (Ctrl+S).

## 3-қадам. Telegram бот (хабарламалар үшін)

1. Telegram-да **@BotFather**-ге жазыңыз → `/newbot` → атын бересіз →
   ол сізге **токен** береді (мыс: `7123456789:AAH4x...`).
2. Өзіңіздің chat ID-іңізді білу үшін **@userinfobot**-қа жазыңыз —
   ол `Id: 123456789` деп жауап береді.
3. Маңызды: өз ботыңызға бір рет **/start** деп жазыңыз (әйтпесе бот сізге жаза алмайды).
4. Apps Script кодындағы `TG_TOKEN` мен `TG_CHAT` өрістеріне осы екі мәнді қойыңыз. Сақтаңыз.

## 4-қадам. Веб-қосымша ретінде жариялау

1. Apps Script редакторында: **Deploy → New deployment**.
2. Түрі: **Web app**.
3. Баптаулар:
   - *Execute as:* **Me**
   - *Who has access:* **Anyone**
4. **Deploy** → рұқсаттарды растаңыз (Advanced → Go to ... unsafe — бұл өз скриптіңіз, қауіпсіз).
5. Берілген **Web app URL**-ды көшіріп алыңыз (`https://script.google.com/macros/s/.../exec`).

## 5-қадам. Сайтты байланыстыру

Репозиторийдегі **`config.js`** файлын ашып, толтырыңыз:

```js
WEBHOOK_URL: 'https://script.google.com/macros/s/СІЗДІҢ_URL/exec',
WHATSAPP_PHONE: '77011234567',   // өз нөміріңіз, тек цифрлар
```

Сақтап, GitHub-қа пуштеңіз (немесе Claude-қа айтыңыз — өзі жасайды).

## 6-қадам. Тексеру

1. Сайттағы форманы толтырып, тапсырыс беріңіз.
2. Telegram-ға хабар келуі керек ✅
3. Google Sheets кестесінде жаңа жол пайда болуы керек ✅
4. `admin.html` бетіне кіріп (сайт мекенжайы + `/admin.html`),
   `ADMIN_KEY` парольді енгізіңіз — тапсырыстар кестесі шығады ✅

### Жиі сұрақтар

**WhatsApp-қа автоматты хабар келе ме?**
Толық автоматты WhatsApp хабар тек ақылы WhatsApp Business API арқылы мүмкін.
Біздің схема: тапсырыс расталғанда клиенттің WhatsApp-ы ашылып, дайын мәтін
сізге жіберуге тұрады — клиент «жіберу» батырмасын басса болды. Telegram-ға
хабар толық автоматты келеді.

**Кестені Excel-ге қалай аламын?**
Google Sheets → File → Download → Microsoft Excel (.xlsx).
Немесе админ-панельдегі «⬇ Excel (CSV)» батырмасы.

**Парольді өзгерткім келсе?**
Apps Script-тегі `ADMIN_KEY`-ді өзгертіп, Deploy → Manage deployments →
Edit → Version: New version → Deploy.
