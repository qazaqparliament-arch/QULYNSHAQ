# Фанер Шахматы · AR

Кітаптағы фигура суретін телефонмен сканерлегенде 3D фигура мен оның жүріс бағыттары
неон болып жанып тұрады. Фигураны телефонды айналдырып, жан-жағынан көруге болады.

## Ішіндегі файлдар
```
index.html            — AR беті (камера + 3D + неон жүрістер)
assets/targets.mind   — 6 маркер компиляцияланған трекинг файлы
markers/              — кітапқа басатын 6 маркер-сурет (PNG)
    king.png   Патша
    queen.png  Уәзір
    bishop.png Піл
    knight.png Ат
    rook.png   Тура
    pawn.png   Сарбаз
```

## Қалай жұмыс істейді
1. `markers/` ішіндегі суреттерді кітаптың сәйкес фигура бетіне басасыз
   (әр фигураға өз суреті — реті index.html ішінде тіркелген).
2. Оқырман телефон камерасын сол суретке бағыттайды.
3. Бет суретті танып, үстінде 3D фанер фигура шығады, ал жүріс бағыттары
   неон сызық болып жанады (Ат — Г-секіріс, Піл — диагональ, т.б.).

## Онлайн орналастыру (МІНДЕТТІ: https қажет)
Камера тек қауіпсіз (`https://`) қосылымда ашылады. Тегін нұсқалар:

**GitHub Pages (ұсынылады):**
1. github.com-да жаңа repository ашыңыз (public).
2. Осы папканың барлық файлын (index.html, assets/, markers/) жүктеңіз.
3. Settings → Pages → Source: `main` branch → Save.
4. Бірнеше минуттан соң `https://<аты>.github.io/<repo>/` сілтемесі шығады.

**Басқа нұсқалар:** Netlify Drop (netlify.com/drop — папканы сүйреп таста),
Cloudflare Pages, Vercel — бәрі тегін әрі https береді.

## Кітапқа QR-код
Орналастырғаннан кейінгі сілтемеге QR-код жасап, кітаптың басына қойыңыз —
оқырман бірден AR бетін ашады.

## Жергілікті тексеру (компьютерде)
```
cd осы_папка
python3 -m http.server 8000
```
Браузерде `http://localhost:8000` ашыңыз (камера localhost-та жұмыс істейді).
Телефоннан тексеру үшін онлайн https сілтеме керек.

## Ескертпелер
- iPhone → Safari, Android → Chrome.
- Маркерді жақсы жарықта, тегіс бетте ұстаңыз.
- Жаңа фигура қосу немесе стильді өзгерту қажет болса — index.html ішіндегі
  `PIECES`, `PROFILES`, `makeMoves` бөлімдерін өңдейміз.
