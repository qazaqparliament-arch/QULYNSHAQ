**IconButton** — a single-glyph action (play, close, nav) or a chess-piece token.

```jsx
<IconButton label="Ойнату" variant="solid" round><span>▶</span></IconButton>
```

Variants: `soft` (amber tint), `solid` (ink), `outline`, `ghost`. Sizes `sm|md|lg`. Set `round` for a circle. Always pass `label` for accessibility.
