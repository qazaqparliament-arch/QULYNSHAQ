**StepList** — numbered how-to steps for assembly and rules (the booklet, on screen).

```jsx
<StepList steps={[
  { title: 'Бөлшектерді басып шығар', body: 'Қатырғаз парақтарын тесіктер бойынша шығар.' },
  { title: 'Стикерлерді жапсыр', body: 'Әр фигураға өз стикерін жапсыр.' },
]} />
```

Amber number discs with ink borders; dashed fold-line dividers (`folds`) echo the buildable box. Steps accept `{title, body}` or plain strings.
