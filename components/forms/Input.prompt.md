**Input** — labelled text field for the order/contact forms.

```jsx
<Input label="Атыңыз" placeholder="Мысалы: Айсұлу" hint="Ата-анаңның аты да болады" />
<Input label="Email" error="Дұрыс email енгізіңіз" leading={<span>@</span>} />
```

Focus ring is teal; `error` turns the border clay and shows the message. Sizes `sm|md|lg`.
